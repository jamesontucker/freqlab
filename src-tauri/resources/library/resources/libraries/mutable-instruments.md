# Mutable Instruments Eurorack

Emilie Gillet's legendary Eurorack module DSP code. MIT License (STM32 only).

## Safe MIT Modules
- **Clouds**: Granular/spectral processing
- **Plaits/Braids**: Oscillators and wavetables
- **Rings/Elements**: Physical modeling
- **Warps**: Frequency shifter/vocoder
- **Stages/Peaks**: Envelopes and LFOs
- **Marbles**: Random/generative
- **Tides**: Complex LFO/envelope

## Warning
AVR-based modules are GPL - avoid those.

## When to Use
- Synthesizer oscillators
- Granular processing
- Physical modeling
- Modulation sources
