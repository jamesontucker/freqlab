# Signalsmith Stretch

Header-only C++11 pitch/time stretching library. MIT License.

## Features
- Polyphonic pitch shifting
- Time stretching
- High-quality results
- Header-only (easy to integrate)

## When to Use
**ALWAYS use this for pitch shifting and time stretching.**

Do NOT use:
- SoundTouch (LGPL)
- Rubber Band (GPL)
