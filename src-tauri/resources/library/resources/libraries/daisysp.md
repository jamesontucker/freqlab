# DaisySP

Electrosmith's DSP library with wide algorithm coverage. MIT License.

## Modules
- **Oscillators**: Multiple waveforms, FM
- **Drums**: AnalogBassDrum, AnalogSnare, HiHat
- **Filters**: SVF and classic designs
- **Effects**: Chorus, Flanger, Phaser, Tremolo
- **Envelopes**: ADSR, AR
- **Utility**: Looper, Portamento

## Warning
Use the main DaisySP repo, NOT DaisySP-LGPL.

## When to Use
- Drum synthesis
- Basic synth building blocks
- Modulation effects
- Learning DSP
