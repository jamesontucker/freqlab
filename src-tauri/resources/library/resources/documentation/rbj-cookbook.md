# RBJ Audio EQ Cookbook

The definitive reference for biquad filter coefficient calculations.

## Filter Types Covered
- Low-pass, High-pass
- Band-pass (constant skirt gain & constant peak gain)
- Notch, All-pass
- Peaking EQ
- Low shelf, High shelf

## When to Use
- Standard EQ implementations
- Simple filter implementations
- DC blocking
